# 南方电网校园招聘年度变化说明

## 1. 总体变化结论

南方电网近5年校园招聘呈现以下趋势：

1. **笔试专业比例下调**：专业知识占比从2022年的70%逐步调整至2026年的40%，综合知识占比从30%升至60%，对行测、企业文化等综合能力的考核权重显著增加。
2. **两批次格局稳定**：每年保持秋招（10-12月）+ 春招（3-5月）两批次，秋招为主、春招补录的模式基本不变。
3. **报名窗口缩短**：从2022年的"公告发布日起、各单位自定"逐步规范为明确的截止日期，2025年起秋招报名期压缩至约2周。
4. **志愿规则趋严**：2025年起投递后有"仅1次撤回机会"，2026年沿用此规则。
5. **环境工程相关岗位方向转型**：岗位关键词从传统"环保管理"转向"碳管理、综合能源、新能源、储能"等方向，纯环保类岗位减少，复合型要求增强。

## 3. 逐年变化说明

### 2026年相较2025年的变化

* **报名时间变化**：秋招报名截止从10月27-31日延至11月2日，略有延长。春招截止3月23日，与2025年相近。
* **笔试内容与分数占比变化**：综合知识占比从40%升至60%，专业知识从60%降至40%。这是近5年最大的一次占比调整，对专业基础薄弱的考生更为友好。
* **笔试时间变化**：秋招笔试从11月下旬调整至12月上旬。
* **春招规则变化**：2026年已参加秋招的可报名春招但不再重复面试（原面试成绩有效）。
* **环境工程相关岗位变化**：2026年储能、新能源方向岗位继续增加，广东电网等下属单位明确设立"碳管理、综合能源"等岗位类型，传统的"环保管理"类岗位进一步收缩。

### 2025年相较2024年的变化

* **报名时间变化**：秋招报名明确为10月14日-27日（约2周），较2024年"各单位不同"更规范。
* **志愿者规则变化**：2025年新增"仅1次撤回机会"的限制。
* **春招规则变化**：2025年春招明确"不沿用上年秋招笔试成绩"，所有资格审查通过者均需重新参加笔试。
* **笔试内容与分数占比**：维持综合40%+专业60%，与2024年一致。
* **招聘对象变化**：2025届招聘对象限定更为严格，不再出现"毕业两年内未就业"等表述。
* **环境工程相关岗位变化**：广东电网等多数下属单位将"环境工程"归入"其他理工类"或"新兴业务类"，独立设置"环保类"岗位的下属单位进一步减少。

### 2024年相较2023年的变化

* **报名时间变化**：2024年各单位报名截止更为分散，最早的11月初、最晚的11月中旬。
* **笔试时间变化**：从11月下旬调整为12月上旬。
* **笔试内容与分数占比**：维持综合40%+专业60%，与2023年一致。
* **招聘流程变化**：2024年新增"南网微招聘APP"渠道，此前仅有PC端系统。
* **环境工程相关岗位变化**：2024年多家单位将环保类专业列入"其他工学类"或"电气类相关专业"，专门列出环保类的岗位减少。

### 2023年相较2022年的变化

* **笔试内容与分数占比变化**：综合知识从30%升至40%，专业知识从70%降至60%。
* **笔试次数变化**：2022年有夏招补录批次（7月），2023年未设立夏招。<br>2023年增设"第二次笔试"时间（3月），给未参加首次笔试的考生第二次机会。
* **招聘流程变化**：2022年允许毕业两年内未就业人员报考，2023年收紧为仅限2023届。
* **环境工程相关岗位变化**：2022-2023年间，南方电网下属部分省公司独立设置"环境工程/环保"岗位，2023年仍有多家单位保留此类岗位设置。

## 4. 对环境工程专业考生的影响

### 可报岗位变化

近5年环境工程专业可报岗位呈现"先稳后减再转型"趋势：
* 2022-2023年，多家省公司设有"环境工程/环保"独立岗位；
* 2024-2025年，独立环保类岗位减少，多并入"其他工学类"或"新兴业务类"；
* 2026年，岗位方向向"碳管理、综合能源、新能源、储能"转型，传统环保岗位进一步收缩。

### 专业备考重点调整

* 2022年专业知识占70%时，环保类专业课程（环境工程原理、水气固废处理等）备考权重高；
* 2026年专业知识降至40%后，投入产出比下降，需平衡综合知识（行测、企业文化）的复习；
* 专业知识考试内容也从"环境工程专业"向"电气+环保综合"方向倾斜。

### 面试与材料准备

* 近两年面试更加注重复合背景：有"环保+电气"交叉项目经历、"碳管理/新能源"相关课题的考生更具优势；
* 纯环境工程背景考生建议在简历中突出能源环境交叉方向项目经历；
* 2025年起面试资格审核更严格，材料不全直接淘汰。

### 综合建议

1. 环境工程专业考生报考南方电网时，优先关注广东电网、南网储能等下属单位的"碳管理、综合能源"类岗位；
2. 笔试备考重点从专业课转向综合知识（行测、企业文化、时政），专业考试补充电气基础常识；
3. 简历中补充能源与环境交叉项目（如碳排放核算、新能源项目环评、储能环保等）；
4. 关注春招补录机会，秋招未上岸仍可参加次年春招。
